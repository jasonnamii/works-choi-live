1

먹는 것.

구조

  • 24/7 채널 = 항상 음악이 필요한 구조

결과
  • 김형석 1,500곡 = "스트리밍 의존" → "방송 인프라 내장 자산" 전환
  • 미국 내 지속 노출 + 재평가 트리거
핵심

IP를 파는 게 아니라 '유통 위치'를 먹는 것.

2
KISAS → "교육이 아니라 유입 파이프라인"
KISAS Funnel

오프라인
수요 생성기.

전제

  • 지상파 = 미국 가정 직접 진입 (특히 학부모)
  • 기존 온라인 마케팅과 완전히 다른 레이어
가능 구조
  • 방송 → 관심 → 캠프·교육 유입
  • 도시별 스테이션 → 설명회·캠프 연결
핵심

채널 = 마케팅이 아니라 "오프라인 수요 생성기".

3
채널 → "창작 데이터 수집 시스템"
Data Factory

데이터
생산 공장.

구조

  • 참여형 포맷(작곡 챌린지 등) 심으면
  • 결과물 = 사용자 생성 창작 데이터
이 데이터의 가치
  • 교육 개선
  • AI 학습 데이터셋
  • IP 후보 발굴
핵심

방송 → 데이터 생산 공장으로 변환.
