# MAD POP 프로젝트 필수 확인 및 요청사항

수신: MADVERSE 및 관련 중국 측 담당자  
발신: Korean Project Team  
일자: 2026년 5월 28일  
프로젝트: MAD POP / 사공이호 1단계

## 1. 요청 목적

MAD POP 프로젝트의 계약 체결과 실행을 위해, 아래 사항에 대한 확인자료와 기재정보를 요청드립니다. 본 요청은 계약 상대방, 지급 주체, IP 사용권자, 지분 또는 옵션 부여 주체, 프로젝트 회사 및 서명권한을 명확히 하기 위한 것입니다.

계약상 당사자는 브랜드명, 프로젝트명 또는 사업명칭이 아니라 실제 법인이어야 합니다. 따라서 `MADVERSE`, `MAD POP` 또는 기타 브랜드명이 계약 당사자로 사용되는 경우, 해당 명칭을 보유하거나 운영하는 실제 법인과 그 대표권을 확인해야 합니다.

## 2. 당사자 및 법인 실체

| 확인 항목 | 요청 내용 | 제출자료 |
|---|---|---|
| 중국 측 지정 계약법인 | 계약 A, 계약 B, 지분 / 옵션 사이드레터에 서명할 실제 법인명 | 법인등기부, 사업자등록 또는 이에 준하는 공식자료 |
| 법인 기본정보 | 설립지, 등록번호, 등록주소, 영업주소, 법정대표자 | 공식 등기자료 |
| 대표권 | 서명자가 해당 법인을 대표하여 계약을 체결할 권한 | 이사회 결의, 위임장, 정관 또는 대표권 증명 |
| 실소유자 | 최종 실소유자 및 지배구조 | UBO 확인서 또는 지배구조도 |
| 브랜드 보유관계 | MADVERSE, MAD POP 명칭과 실제 법인의 관계 | 상표등록, 브랜드 사용권 또는 운영권 자료 |

## 3. 프로젝트 회사 및 지분 또는 옵션 대상 회사

| 확인 항목 | 요청 내용 | 제출자료 |
|---|---|---|
| 프로젝트 회사 | MAD POP 프로젝트를 운영할 법인 | 법인등기자료, 정관, 주주명부 |
| 지분 또는 옵션 대상 회사 | KHS Holder가 30% 지분 또는 30% 지분 옵션을 취득할 회사 | cap table, 정관, 주주계약, 전환증권 내역 |
| 완전희석 기준 | 옵션, 전환증권, SAFE, warrant, ESOP 포함 여부 | 완전희석 cap table |
| 기존 투자자 권리 | 우선주, 우선청산권, 희석방지권, 동반매도권 등 | 투자계약 및 주주계약 |
| 상장 구조 | 홍콩, 런던 또는 기타 시장 상장 추진 구조 | 상장 로드맵, 자문사, 예상 일정 |

## 4. 지급 및 자금 관련 확인

| 확인 항목 | 요청 내용 |
|---|---|
| 계약 A 대가 지급 주체 | KHS 크리에이티브 회사에 크리에이티브 피를 지급할 법인 |
| 계약 B 사용료 지급 주체 | 노느니에 라이선스 피 및 러닝 로열티를 지급할 법인 |
| 송금 경로 | 송금 국가, 송금은행, 통화, 세무 원천징수 방식 |
| 원천징수 | 중국 또는 기타 관할에서 원천징수되는 세금과 증명서 제공 방식 |
| 투자금과 분리 | 계약 A 대가 및 계약 B 사용료가 프로젝트 투자금, 주주대여금, 비용상환금, 운영비 또는 출자금으로 재분류되지 않는다는 확인 |
| 최소 투자금 | 중국 측이 MAD POP 프로젝트에 투입할 최소 투자금, 지급일정 및 미지급 시 효과 |

## 5. 사공이호 IP 계약 원칙 확인

| 확인 항목 | 확인 내용 |
|---|---|
| 원천 IP 소유권 | 사공이호 원천 IP의 소유권은 이전되지 않는다는 원칙 |
| 승인 범위 | 사용 가능한 자료와 목적은 권리자가 승인한 범위로 한정한다는 원칙 |
| 별도 승인 대상 | 승인 범위를 넘는 확장 사용, 제3자 사용 허락, 파생 사용은 별도 서면 합의가 필요하다는 원칙 |
| 금지행위 | 승인 없는 등록, 권리 이전 주장, 담보 제공, 금융상품화, 원천 IP와 혼동되는 별도 권리화 금지 |
| 정산·감사 | 사용료, 러닝 로열티, 매출보고, 회계자료 제공 및 감사권의 기본 구조 |
| 정지·회수 | 미지급, 무단 사용, 브랜드 훼손, 회계자료 미제공 시 사용 정지 또는 회수 기준 |

| 권리 확보 | 초상권, 성명권, 음성권, 퍼블리시티권, 저작권, 저작인접권 동의 확보 방식 |
| 디지털 관련 제3자 권리 | 초상, 성명, 음성, 이미지, 저작권 등 제3자 권리 확보 방식 |
| 확정 리소스 | 이미 계약된 리소스와 협의 중인 리소스 구분 |
| 투자자 설명 | 미확정 제3자 리소스를 투자자, 거래소, 기관 또는 미디어 자료에 표시할 계획 여부 |

## 10. 확인의 효과

위 자료가 확인되기 전에는 계약 당사자, 지급 주체, IP 사용권자, 지분 또는 옵션 부여 주체 및 책임 범위를 최종 확정하기 어렵습니다. 양측은 위 확인자료를 바탕으로 계약 A, 계약 B 및 지분 / 옵션 사이드레터의 최종 서명본을 확정합니다.
