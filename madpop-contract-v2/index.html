01 / 개요

V2는 실제 계약으로 이어지기 위한 정리본입니다.

V1이 방향을 설명한 문서였다면, V2는 상대방에게 보내고 변호사가 검토할 수 있도록 정리한 문서입니다. 핵심은 계약 상대, 돈을 지급할 주체, IP 권리자, 지분 또는 옵션을 받을 주체를 서로 섞지 않는 것입니다.

핵심 방향
한 장짜리 JV 계약으로 처리하지 않습니다.

창작 업무, 사공이호 IP 사용권, 30% 지분 또는 옵션은 성격이 다릅니다. 그래서 하나의 계약서에 모두 넣지 않고 문서를 나눕니다.

상대방에게 설명할 방향
거절이 아니라 구조를 명확히 하자는 뜻입니다.

중국 측에는 조건을 낮추자는 얘기가 아니라, 실행 과정에서 문제가 생기지 않도록 책임과 돈의 흐름을 분명히 하자는 취지로 설명합니다.

04 / 계약서

계약 A와 계약 B는 서로 다른 권리를 다룹니다.

김형석의 창작 업무와 노느니의 사공이호 IP는 같은 자산이 아닙니다. V2에서는 두 권리를 다른 계약으로 나누고, 돈의 흐름도 따로 정리합니다.

항목V2 내용필요한 이유
당사자중국 측 계약 법인과 KHS 크리에이티브 회사김형석 개인에게 책임이 과도하게 넘어가는 것을 막기 위해서입니다.
서비스 범위크리에이티브 디렉션, 총괄 프로듀싱, 음악감독, 신규 IP 개발, 현지화 방향 제시상장 성공, 투자유치, 인허가, 제3자 섭외까지 보장하는 계약으로 보이지 않게 하기 위해서입니다.
대가계약 체결, 창작 방향 확정, 현지화·신규 IP 작업, 런칭 준비 단계별로 나눠 지급작업 단계와 지급 시점을 맞추기 위해서입니다.
책임 제한고의 또는 중과실, 직접손해, 수령금 한도를 기준으로 책임 제한기대이익, 사업손실, 상장 지연 손해까지 책임지는 구조를 피하기 위해서입니다.
항목V2 내용필요한 이유
사용권지역, 기간, 프로젝트, 승인 자료, 승인 목적을 정한 제한적 사용권사공이호 원천 IP가 넘어간 것으로 해석되지 않게 하기 위해서입니다.
제외 권리원본 파일, 음악, 제3자 권리, 별도 승인 없는 확장 사용, 한국 내 권리는 제외허락하지 않은 방식으로 IP가 확장 사용되는 것을 막기 위해서입니다.
로열티사용료, 러닝 로열티, 제3자 사용 허락 수익에 대해 별도 정산사공이호 IP에서 실제 매출이 발생할 경우 노느니가 후속 수익을 받을 수 있게 하기 위해서입니다.
회수권미지급, 무단 사용, 별도 승인 없는 확장 사용, 브랜드 훼손, 회계자료 미제공 시 사용 정지 또는 해지문제가 생겼을 때 IP 사용을 멈추거나 회수할 수 있게 하기 위해서입니다.
항목기준의미
계약 A 대가KRW 4,500,000,000~4,700,000,000창작 업무에 대한 대가로 구분합니다.
계약 B 사용료KRW 300,000,000~500,000,000사공이호 IP 사용에 대한 선급금 또는 최소보장 사용료입니다.
돈의 성격 고정투자금, 주주대여금, 운영비, 상장비용, 출자금으로 바꿔 해석하지 않음상대방의 비용 처리와 한국 측 대가가 섞이지 않게 하기 위해서입니다.
분쟁해결홍콩법/HKIAC 우선, 싱가포르법/SIAC 2순위분쟁이 생겼을 때 중립적인 절차로 처리하기 위해서입니다.
05 / 지분·옵션

30% 권리는 KHS Holder가 받는 구조로 정리합니다.

노느니는 사공이호 IP 권리자 역할에 집중합니다. 지분 또는 옵션은 KHS Holder가 받는 구조로 둡니다. 직접 지분 보유가 어렵다면 옵션, 팬텀 지분, 현금정산형 성공보수로 바꿀 수 있습니다.

구조 내용 적용 상황
직접 지분KHS Holder가 대상 회사의 30% 지분을 직접 보유법률·세무상 직접 보유가 가능할 때
옵션KHS Holder가 대상 회사의 30% 지분을 취득할 수 있는 옵션 보유바로 지분을 받을 수 없거나 조건을 붙여야 할 때
팬텀 지분30% 지분 가치에 연동한 현금정산권 보유직접 지분이나 옵션 실행이 어려울 때
현금정산형 성공보수매각, 상장, 투자유치, 지배권 변경 시 30% 경제가치 정산공직, 외환, 규제 이슈 때문에 직접 보유가 어려울 때
30% 권리 보호 기준
  • 30%는 완전희석 기준으로 계산합니다.
  • 보통주, 우선주, 전환사채, SAFE, warrant, option, ESOP, 자문사 보상지분까지 포함해 계산합니다.
  • 신주발행, 정관변경, 특수관계거래, IP 처분, 상장 구조 변경은 사전 동의가 필요한 주요사항으로 둡니다.
  • 직접 지분이나 옵션이 어렵다면 30%에 해당하는 경제적 가치를 현금정산권으로 보전합니다.
06 / 법무

변호사에게는 핵심 쟁점만 나눠 검토를 요청합니다.

V2 법률 검토 요청서는 이 구조가 실제로 서명 가능한지, 나중에 집행 가능한지 확인하기 위한 문서입니다. 세무, 외환, IP, 중재 리스크도 함께 점검합니다.

우선순위 쟁점 확인할 내용
1계약 당사자와 대표권브랜드명이나 프로젝트명이 아니라 실제 계약 법인을 특정할 수 있는지, 대표권과 최종 지배구조 확인이 충분한지 봅니다.
2김형석 개인책임김형석을 핵심 창작자로 명시하는 문구가 개인보증이나 개인 배상책임으로 확대되지 않는지 봅니다.
3IP 라이선스원천 IP, 승인 자료, 제외 권리, 별도 승인 없는 확장 사용 금지, 회수 조항이 충분한지 봅니다.
430% 지분 또는 옵션완전희석 기준, 희석방지, 정보권, exit, 백지신탁 대응 조항이 실제로 작동할 수 있는지 봅니다.
5세무·외환한국, 중국, 홍콩 또는 싱가포르 기준으로 원천징수, 외환, 해외직접투자 신고 이슈를 확인합니다.
6준거법·중재홍콩법/HKIAC 또는 싱가포르법/SIAC 구조가 적절한지, 중국 내 집행 가능성은 어떤지 봅니다.
