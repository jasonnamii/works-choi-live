#3 · Solution

8개 기능.
그게 전부다.

3면의 20개 Pain은 하나의 부재로 수렴한다 — "일 찾기 → 연결 → 관계 축적" 여정을 구조화한 전용 플랫폼이 없다.

①

프로필 + 협업이력

포트폴리오(실력) 위에 협업이력·평판·관계 데이터를 쌓는다. "한번 해봐야 안다" → "프로필에서 미리 본다".

②

홍보 / 브랜딩

프로필 생성 즉시 Hiring Side·S&E에 노출. 기다리는 게 아니라 기회가 찾아오는 구조.

③

검색

크리에이티브의 비구조화 정보(음색·연출력·케미)를 검색 가능한 데이터로 변환.

④

추천 / 매칭 + 팀

1:1 매칭 + 검증된 팀 조합 저장·재소집. 솔로→팀→스튜디오 성장 경로의 인프라.

⑤

프로젝트 공고

비공식 62%+ 채널에 갇힌 기회를 플랫폼 위로. 단발이 아닌 관계 기반 반복 협업으로 연결.

⑥

접촉 도구

카톡 번호 없이, 지인 없이, 플랫폼 안에서 연결 완결.

⑦

단가 레퍼런스

직군×경력×지역별 시세. 크리에이티브 전 직군 커버하는 시세 기준 현재 0건.

⑧

풀 관리

62%가 엑셀로 추적하는 프리랜서 풀을 시스템화. 개인 인맥 → 조직 자산 전환.

#4 · Why Now

가능해졌고,
급하고, 비어 있다.

  2. ② 비용 1,000×↓ (3년)

  4. ④ 공급 임계점 (US 프리랜서 7,000만)
  5. ⑤ 수요측 지불의사 (관리 SW $14.8B)
  6. ⑥ 제도 인프라 (FIFA·CA·NJ 서면계약)

  1. ① 갭 가속 — E&M $3T CAGR 3.7%
  2. ② 데이터 방어벽 윈도우 (2026)
  3. ③ AI 범용화 데드라인 (MMLU 17.5pp→0pp)
  4. ④ 경쟁자 진입 윈도우

이것이 #3 ⑧풀 관리의 실체이자 Creative Graph의 시작점. 프로필(누구) + Respect(얼마나 신뢰) + 관계(누구와 일했는가) = "누가 누구와 어떻게 일했는가"가 데이터로 쌓인다.

#7 · Business Model

도구에 과금한다.

Creator는 무료로 발견되고, 찾는 쪽이 도구를 구독한다. 매칭 "성사"가 아니라 매칭 "도구"에 과금. 구독 + 광고 + 솔루션, 3개 엔진.

왜 도구 과금인가

반복 협업이 자기잠식되지 않으려면.

거래 성사 과금

Fiverr 27.6% · Upwork 18.5%

반복 협업 → 플랫폼 우회 → 매출 감소

도구 사용 과금

LinkedIn $17.8B(₩24.6조) Revenue

반복 협업 = 도구 사용 ↑ = 매출 ↑

크리에이티브 산업은 단건 완결이 아니라 반복 협업이다. 거래에 과금하면 관계 축적(#6 ③)이 자기잠식. 도구에 과금해야 네트워크가 강화된다.
