05 · Budget Architecture

2억 = "광고비"가 아니라
"씬 점화 자금".

7개 항목은 따로 노는 라인이 아니라 하나의 퍼널. 콘텐츠 → 광고 → 유입 → 포트폴리오 → 관계 → 재방문 → 초대.

₩2억
TOTAL · 7 LINES

광고비가 35%일 뿐, 나머지 65%는 씬을 만드는 비용.

35% 7,000만 Meta 퍼포먼스 설치·포트폴리오 업로드 핵심

12.5% 2,500만 콘텐츠 제작 월 50~100개 광고 공장

5% 1,000만 CRM / 리텐션 푸시·이메일·하이라이트
7.5% 1,500만 데이터 / 실험 버퍼 Hook·길이·타겟·카테고리·AI 5축 AB 테스트 — "실패하기 위한 돈"

09 · Asset Activation

자산은 4개 — 호출 맥락은 명확히 분리.

같은 자산이 여러 트랙에서 다른 의미로 쓰이면 메시지가 흐려집니다. 자산별 역할 1개 + 호출 트랙 1~2개로 고정.

ASSET 01

김형석 작곡가

감각·취향의 상징 · "색은 따라할 수 없다"

호출 트랙: Track B(Pick Challenge) + Track C(Wanted 밴드 모집). 일반 광고 ✗ · 산업 진입의 상징으로만.

ASSET 02

영상 제작사 네트워크

결과물 비주얼 · MV Collaboration

호출 트랙: Track A(콘텐츠 제작 소재) + Track B(시드 결과물 가시화). UI 광고 ✗ · "멋진 결과물"로.

ASSET 03

댄스 연결

성장 엔진 · 숏폼·바이럴·글로벌

호출 트랙: Track A(TikTok 메인) + Track B(Cover Team Match). "춤 자체"보다 "팀 결성 장면".

ASSET 04

팝아트 레이어

세계관 자산 · 굿즈·비주얼·챌린지

호출 트랙: Track C(Wanted 비주얼) + 브랜딩 굿즈. 메인 acquisition ✗ · 세계관 강화로만.

10 · Phase Roadmap

씨앗 → 집중 → 폭발. 3페이즈 구조.

Seed

0 ~ 5,000만 · 4주

  • 50~100개 소재 빠른 실험
  • Hook·길이·타겟·카테고리 AB

  • Wanted 1차 성수 5~10곳
  • 대행사 학습 페이즈 — 빠른 폐기

Concentrate

5,000만 ~ 1.5억 · 6주

  • 먹히는 카테고리 집중 투입
  • 김형석 Pick Challenge 1차
  • 72시간 K-pop Team Challenge 1회
  • MV Collaboration 1팀 가시화
  • Wanted 2차 성수 + 홍대 확장

GOAL · 시드 활동 유저 → 자생 콘텐츠 점화

PHASE 03

Network Explosion

1.5억 ~ 2억 · 4주

  • 챌린지·팀매칭·협업 이벤트 강화
  • 리퍼럴 보상 풀가동
  • Wanted 3차 — 건대·이태원·강남 확장
  • 실제 결과물(밴드·MV·곡) 송출
  • Phase 4 시리즈A 스토리 빌드

GOAL · 네트워크 효과 임계점 돌파
