# 기능정의 리스트 — 갤러리

> Cre8orClub 두 번째 메뉴 "갤러리". 숏폼의 반대인 롱폼으로, 한 시간에 단 하나의 작품을 모두가 같은 시간 축으로 보는 라이브 전시입니다.
> 0.1(컨셉 초안)에 결정 12건을 반영한 0.2입니다. 정책·예외가 대부분 확정됐고, 화면 자료가 필요한 3건만 제안 반영(디자인 확정 대기)입니다.

## 상태 요약

- **가는중 8** (F-001·F-002·F-003·F-004·F-005·F-006·F-007·F-009·F-010 중 9개) — 결정 반영으로 규칙이 확정돼 착수 가능. *(F-010 신규 포함)*
- **못감-정보부족 1** (F-008) — 예고편 형식은 영상 트레일러로 제안했으나 디자인·제작 여건 확정이 필요합니다.
- 화면 자료 대기(제안 반영): 빈 시간(F-001/F-009)·재생 오류(F-002)·예고편(F-008) — 디자인 확정 시 갱신.

> 확신도가 0.1(70~80)보다 오른 것은 컨셉 추정에 **사람의 결정**이 더해졌기 때문입니다(80~88). 화면 확인이 남은 F-008은 74에 머뭅니다.
