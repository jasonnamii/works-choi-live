## 🔴 먼저 정할 것 (가장 큰 갈림길)
- **D-1** 한 시간 지난 작품은 사라지나 남나 — 갤러리 정체성을 가르는 한 수
- **D-2** 한 시간 한 작품을 누가·어떻게 정하나 — 편성·검수 동선 전체가 갈림
- **D-6** 댓글을 v1에 넣나 — 넣으면 모더레이션·신고 운영이 따라옴

## ⚠️ 헤비 검토 권장 (위험 기능 — 참고)
신고·차단·UGC(남의 작품) 노출이 들어 있어 정책·예외를 꼼꼼히 설계하길 권합니다.
- **댓글**(F-006) — 모더레이션·신고·차단 정책 필요. 운영 부담 큼.
- **작품 노출**(F-002·F-009) — 남의 창작물을 한 시간 노출 → 이용허락·저작권 확인 필요.
- 관련 전문: 약관·신고 동선 → `app-and-jang` / 저작권·UGC 귀속 → `ip-skill`

## ⚖️ 법무 확인 (생성 ✗ · 라우팅만 — 변호사 최종검토)
- 작품 노출의 **이용허락 범위**(기간·재노출·보관) → `app-and-jang` / `ip-skill`
- 작품 속 **사람 얼굴·음성 초상권** 동의 범위 → `app-and-jang`
- 작품이 **AI 생성물일 때 표시** 필요 여부 → `app-and-jang`
- 댓글·작품 **권리침해 신고 시 임시조치(즉시 내림)** 의무 → `app-and-jang`
