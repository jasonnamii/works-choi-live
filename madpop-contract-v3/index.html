중국 측에는 문서 5종을 한 번에 보냅니다.

이번 발송본은 커버레터, 마스터 텀시트, 크리에이티브 디렉터 계약, 사공이호 IP 라이선스 계약, 지분계약으로 구성됩니다. 메시지는 긍정적인 파트너십 제안으로 잡고, 권리·금액·사용 범위·확인 자료는 문서 안에서 명확히 정리했습니다.

01

송부 방향

  • 한 번에 전달. 중국 측에는 문서 5종 전체를 같은 메일에 첨부해 보냅니다.
  • 첫인상은 파트너십. 커버레터는 협업 의지와 실행 기대를 먼저 보여줍니다.
  • 조건은 문서에서 명확화. 금액, 사용 범위, 지분, 확인 자료는 각 문서에 빠짐없이 넣었습니다.
  • 권리는 분명하게 유지. 사공이호 IP와 김형석 PD님의 기존 저작권은 권리자에게 계속 귀속됩니다.
  • 추가 제작은 별도 합의. 출장, 녹음, 편곡, 영상, 공연, 신규 IP 개발은 별도 프로젝트로 진행합니다.
  • 서명 전 확인. 상대 법인과 사업 실행 기반을 확인한 뒤 지급, 사용권 개시, 지분 반영을 진행합니다.
02

중국 측 확인 자료

문서 5종에는 아래 자료 제출을 서명 전 확인 절차로 넣었습니다. 자료가 확인되면 지급 일정, 사용권 개시, 지분 반영 절차를 같은 기준으로 진행할 수 있습니다.

  • MADVERSE 측 법인등기·실질지배주주(UBO)·그룹 조직도
  • 최근 3개년 재무제표 · 12개월 사업비 · 자금조달 계획
  • AI 모델 자체개발 증빙 또는 외부 라이선스 계약
  • 방영 플랫폼·유통망 계약(Bilibili 포함) · SNS·배급망 현황
  • UMG · 咪咕 · 龚廷海 페스티벌 IP 협력 현황
  • 현물 출자 가치 산정 · cap table · 정관

확인 자료는 양측이 같은 일정표로 계약 효력, 지급, 사용권 개시, 지분 반영을 진행하기 위한 공통 체크포인트입니다.
