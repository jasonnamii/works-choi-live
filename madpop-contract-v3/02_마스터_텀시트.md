# 02. 마스터 텀시트 — MAD POP / 사공이호 (V3)

일자: 2026년 6월 · 통화: USD 기준, KRW 병기(환율 가정 1,380) · 성격: 협의용 비구속 문서(비밀유지·독점협상 조항은 구속)

> 본 텀시트는 크리에이티브 디렉터 계약, 사공이호 IP 라이선스 계약, 지분계약을 하나의 실행 구조로 연결합니다. 실제 권리와 의무는 각 본계약에서 확정합니다.

## 0. 당사자

| 구분 | 내용 |
|---|---|
| MADVERSE 측 | MADVERSE가 지정하는 실제 계약법인 (법인명·설립지·등록번호·대표권·UBO 제출 필요) |
| 김형석 측 | 김형석 PD 또는 지정 KHS 크리에이티브 회사, 노느니 또는 사공이호 IP 권리자, KHS Holder |
| 계약 원칙 | 브랜드명이 아니라 실제 법인을 계약 당사자로 두고, 역할·IP·지분을 각각의 전문 계약으로 구성 |

## 1. 거래 구조

김형석 PD는 MAD POP의 기획·프로듀싱에 참여하고, 사공이호 IP는 중국 사업 실행에 필요한 범위에서 라이선스하며, KHS Holder는 합자회사 지분 30%를 보유합니다. 세 권리는 서로 다른 계약으로 정리하되, 프로젝트 실행 일정과 지급 구조는 하나의 패키지로 관리합니다.

## 2. 크리에이티브 디렉터 계약 (요약)

| 항목 | 조건 |
|---|---|
| 역할 | 기획 자문·음악/콘텐츠 방향성 프로듀싱·승인된 명칭 사용 (경영·재무·상장 보증은 별도 대상 아님) |
| 기간 정액 | **USD 3.5M (약 48억 원)** |
| 단계지급 | 체결 40% / 창작방향 패키지 30% / 현지화·개시 20% / 런칭검토 10% |
| 프로듀서 포인트 | 대상 매출의 **5%** |
| 추가 업무 | 출장·녹음·편곡·영상·공연·투자자료 협조·신규 IP 개발은 별도 프로젝트 계약으로 진행 |
| 명칭 사용 | 사전 승인된 목적·매체·기간 안에서 사용 |

## 3. 사공이호 IP 라이선스 계약 (요약)

| 항목 | 조건 |
|---|---|
| 부여 권리 | 제한적·양도불가·재사용허락 불가 사용권 |
| 선급 최소보장(MG) | **USD 1.5M/년 (약 21억 원)**, 비환불·러닝에서 recoup |
| 러닝로열티 | 직접사용 6% / 상품화 12% / 서브라이선스 순수령 35% |
| 대상 IP | 별첨으로 승인한 사공이호 기존곡·기존 캐릭터. 신규곡·신규 캐릭터·확장물은 별도 프로젝트 계약 |
| 기간·연장 | **1년**, 연장은 별도 서면 연장계약으로 진행 |
| 영역(Territory) | **중국 한정** |
| 종료·갱신 | 사업 운영, 지급, 보고, 승인 범위 준수 여부에 따라 갱신 또는 종료 |

## 4. 지분계약 (요약)

| 항목 | 조건 |
|---|---|
| 지분 | 합자회사 **30%** (완전희석 기준), KHS Holder 보유 |
| 주요사항 동의권 | 신주발행·정관변경·특수관계 거래·IP 처분·청산·M&A·CEO 선임 등 |
| 보호장치 | 희석방지(weighted average)·청산우선·정보권·감사권 |
| Exit | 동반매도권·강제매도 제한·우선매수권·**풋옵션**(36개월·자금 미투입·IPO 미진행·추가 제재 시) |
| 운영 책임 | 김형석 측은 경영·재무·상장·시장성과에 관한 보증을 제공하지 않음 |
